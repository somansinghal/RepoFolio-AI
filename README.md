# ⚡ Features

### 👤 Developer Profile Dashboard

Displays:

* GitHub avatar
* followers
* following
* public repositories
* total stars

---

### 📊 Language Analytics

Interactive chart showing language distribution using Chart.js.

Example:

JavaScript
Python
HTML
CSS

---

### 🏆 Developer Score

A custom scoring system based on:

* GitHub followers
* repository count
* star count
* activity level

---

### 📈 GitHub Insights

Shows:

* contribution graph
* GitHub streak stats
* top languages
* top repository

---

### 🔎 Repository Explorer

Users can:

* search repositories
* sort by stars
* sort by name

---

### 🧠 Skill Detection

Automatically detects developer skills from repositories.

Example output:

JavaScript
Python
Machine Learning
React

---

### 🌍 Shareable Portfolio

Generate shareable portfolio links.

Example:

```id="ybbvll"
https://repo-folio-ai.vercel.app/?user=somansinghal
```

---

### ⬇️ Portfolio Download

Download a standalone HTML version of the generated portfolio.

---

### 🎨 Theme System

Choose between multiple themes:

* Dark Dev
* Glass
* Cyberpunk
* Minimal
* Gradient

# 🔑 GitHub API Setup

Generate a personal access token on GitHub.

Add inside **script.js**

```id="6i23oe"
const GITHUB_TOKEN = "your_token_here"
```

API limits increase from:

```id="r4sio6"
60 requests/hour → 5000 requests/hour
```
